Principales caracteristicas de CSS3

  • MÓDULO DEL COLOR CSS3:
    • CSS3 soporta más color y una gama más amplia de las definiciones del color. Los nuevos colores que proporciona CSS3 es HSL, CMYK, HSLA y RGBA.
  • ESQUINAS REDONDEADAS O BORDES REDONDEADOS:
    • Crea fácilmente las esquinas redondeadas para cualquier caja usando border-radius y background position.
  • CAJAS CON SOMBRA:
    • Box-shadow permite agregar un efecto de sombra sin usar imágenes a un elemento seleccionado. Box-shadow es soportado actualmente del Safari 3+ y Firefox 3.1+.
  • MÚLTIPLES COLUMNAS (MULTIPLE COLUMNS):
    • Este módulo de CSS3 permite colocar los textos en varias columnas de forma mucho más simple usando las propiedades: -moz-column-count and -moz-column-width.
  • SOMBRA PARA TEXTO:
    • text-shadow de CSS3 permite agregar una sombra a cada letra del texto. Esta propiedad no es nueva en CSS3, fue propuesto originalmente en CSS2, pero quitado en CSS 2.1.

CSS Responsive

RWD

Proveniente de sus siglas Resective Design Web (Diseño Web Receptivo).
La razón principal para mantener su diseño receptivo es aumentar el alcance de su aplicación a una base de usuarios más grande utilizando una variedad de dispositivos

Tecnologias modernas

Existen métodos modernos de diseño de páginas web que asumen que tratas
de crear una cuadrícula flexible y te proporcionan los modos más fáciles de hacerlo.

  • Multicol
  • Flexbox
  • Grid CSS

Tipos de diseño receptivo

  1. Adaptado

    2. Entregarán diferentes sitios web según el tipo de dispositivo que visite el sitio.
    Esto significa que el sitio móvil será totalmente
    diferente de (y separado de) el escritorio.

    Es decir:
    3 sitios web / Tipo de dispositivo / Cambia el contenido






  2. Responsive

    3. Es una forma de crear sitios web, por lo que el contenido se reorganiza según el tamaño de la pantalla.
    Puede saber si un sitio responde cambiando el tamaño de la ventana de su navegador.
    ¿El contenido se reorganiza a sí mismo? Si es así, entonces el sitio responde.

    Es decir:
    1 sitio web / Tamaño de pantalla / No cambia contenido
